## ADDED Requirements

### Requirement: 动态插件通过密钥引用解析宿主敏感凭证

系统 SHALL 为动态插件提供受治理的密钥解析服务，插件只能通过宿主授权的密钥引用解析敏感凭证，不能直接读取宿主配置文件或明文密钥。

#### Scenario: 插件解析授权密钥引用

- **WHEN** 插件调用密钥服务解析一个已授权的`host-secret`资源
- **THEN** 宿主返回该资源允许暴露的密钥内容或派生凭证
- **AND** 宿主对返回结果执行最小必要暴露控制

#### Scenario: 插件尝试读取未授权密钥

- **WHEN** 插件调用一个未授权的密钥引用
- **THEN** 宿主拒绝该调用
- **AND** 宿主不向 guest 暴露宿主敏感配置结构
