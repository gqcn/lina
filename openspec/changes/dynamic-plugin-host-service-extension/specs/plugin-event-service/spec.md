## ADDED Requirements

### Requirement: 动态插件通过命名事件主题发布宿主事件

系统 SHALL 为动态插件提供受治理的事件服务，插件只能向宿主授权的事件主题发布事件，而不能直接操作宿主内部事件总线实现。

#### Scenario: 插件发布授权主题事件

- **WHEN** 插件调用事件服务向已授权的`host-event-topic`发布事件
- **THEN** 宿主校验主题权限、事件大小和事件元数据
- **AND** 宿主将事件投递到对应主题

#### Scenario: 插件发布未授权主题事件

- **WHEN** 插件尝试向未授权主题发布事件
- **THEN** 宿主拒绝该调用
- **AND** 宿主记录拒绝摘要用于审计
