## ADDED Requirements

### Requirement: 动态插件通过命名队列提交异步任务

系统 SHALL 为动态插件提供受治理的队列服务，插件只能向宿主授权的命名队列提交异步任务。

#### Scenario: 插件向授权队列提交任务

- **WHEN** 插件调用队列服务向已授权的`host-queue`提交任务
- **THEN** 宿主校验队列权限、消息大小和任务元数据
- **AND** 宿主返回任务投递结果或任务标识

#### Scenario: 插件尝试向未授权队列提交任务

- **WHEN** 插件向未授权队列提交任务
- **THEN** 宿主拒绝该调用
- **AND** 插件不能绕过宿主异步任务治理边界
